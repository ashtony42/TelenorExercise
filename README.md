Hello! Welcome to my Telenor exercise!

- DotNetSeleniumExtras.PageObjects.Core
- Gherkin
- NUnit
- Selenium.Support
- Selenium.WebDriver
- SpecFlow
- SpecFlow.Assist.Dynamic
- SpecFlow.NUnit
- SpecFlow.Tools.MsBuild.Generation
- WebDriverManager

**RUN**
